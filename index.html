あなたの人生、残りどれくらい？

限りある自由な時間。
平均的な睡眠時間(7時間)と寿命(84歳)を元に、
あなたに残された貴重な時間を計算します。

または

この時間で、絶対にやりたいこと

具体的な期限を決めると、夢はぐっと近づきます。
例えばこんなふうに、期間と結びつけてみませんか？

「好きなこと」はなくていい。「やりたいこと」に変換すればいい

「好きなことがないんです」
「何をやりたいか、ずっとわからない」
そんな声をよく聞きます。でも本当は、“好き”を見つけることがゴールじゃないんですよね。

好きなことより「やりたいこと」のほうが見つけやすい理由

「好き」は感情ベース、「やりたい」は行動ベースです。
たとえば、

  • 本を読むのが“好き” → 書評ブログを“やってみたい”
  • 散歩が“好き” → 地元の観光マップを“作ってみたい”
  • 誰かの話を聞くのが“好き” → コーチングを“学んでみたい”

こんなふうに、「好き」を“動詞化”すれば「やりたいこと」に変換できます。
大事なのは、「今の自分」で完璧に見つけようとしないこと。

それでも見つからない人へ：「場所を変える」という選択肢

同じ環境、同じ人間関係、同じSNS。
これで何か新しい“ひらめき”を期待するのは、正直ムリがあります。
だから、まずは“場所”を変えてください。

  • 日帰りでもいい、旅行に行く
  • 引っ越しを前提に物件を見てみる
  • カフェや図書館で読書する
  • あるいは本当に、1冊の本と出会うだけでもいい

思考と場所を変えてくれる5冊の本

1. 『エッセンシャル思考』グレッグ・マキューン

「より少なく、しかしより良く」。
常識にとらわれず、自分にとって本当に重要なことを見極める力が養えます。
→ 「全部やろうとして何も進まない」人に。
詳細を見る

2. 『LIFE SHIFT』リンダ・グラットン

100年時代の人生戦略。
→ 「この会社でこのまま定年まで？」と悩んでいるあなたに、人生を再設計するヒントを。
詳細を見る

3. 『深夜特急』沢木耕太郎

旅を通じて「自分」を見つけた著者のリアル。
→ 一歩踏み出したいけど怖い、そんな時の背中を押してくれる一冊。
詳細を見る

4. 『やりたいことの見つけ方』八木仁平

自己分析で「迷子の自分」に言語を与える。
→ 行動できないまま悩んでいる人に理論で光を。
詳細を見る

5. 『今日の人生』益田ミリ

丁寧に暮らす日々の中に、“豊かさ”が宿る。
→ 「やりたいことがなくても、別にいいかも」と思わせてくれる本。
詳細を見る

やりたいことは、「あなたの外」にある

やりたいことは、頭の中でうんうん考えても見つからない。
動いた先、出会った人、ふと読んだ一冊の中に、“変な引っかかり”として見つかるものです。

「好きなことが見つからない」は、終わりじゃない。
むしろ、“やりたいこと”が見つかるスタート地点です。
